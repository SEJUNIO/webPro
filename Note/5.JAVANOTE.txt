[배열] :데이터를 관리해 주는 중요한 요소
ㄴ동일 자료형의 집합. 하나의 이름으로 여러개의 데이터를 사용
ㄴ하나의 데이터만 가지고 있던 변수와 달리 여러 개의 데이터를 저장한 곳의 주소
배열이란 : 배열문법, 배열의 메모리 구조, 레퍼런스의 이해, 배열복사, 다차원배열
--배열의 문법--
여러 개의 데이터가 모여 있어 {}를 이용하여 초기화
배열의 크기는 최초 한번 설정되면 변경 불가
배열을 객체로 취급하며
배열선언 -> 배열의 메모리 할당 -> 배열이용
ex / int i =10;
	int[] iArr = {10,20,30,40,50};  //선언과 초기화를 한번에
변수와 같이 이해하는 배열 메모리 구조 이해
int i =10;  // int라는 메모리 에 i 주머니가 만들어 지고, i 주머니는 10 이라는 데이터가 들어있다.
int[] iArr = {10,20,30]; 메모리에 iArr 주머니가 만들어지고 iArr주머니 안엔 배열을 구성하고 있는 데이터 주소값이 들어있다.

배열은 배열을 구성하고 있는 데이터들의 주소값을 가지고 있다 하였으며
이 주소값을 "레퍼런스"라고 한다.
만약 동일한 주소값을 가지고 있다면, 같은 데이터를 가리키고 있다 할 수 있음.
