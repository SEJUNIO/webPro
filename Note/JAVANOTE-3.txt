[연산자]
일정한 규칙을 가지고 계산하는 것으로 일상생활에서 사용하고 있는 덧셈, 뺄셈, 곱셈, 나눗셈을 비롯하여 프로그램에서는 자동증감, 동등비교, 비트연산 등 많은 연산자가 있다
연산자의 종류는 : 산술, 증감연산, 동등관계, 논리연산, 삼항연산, 할당연산자 가 있다.
산술 연산자 : * / + - %
증감 연산자 : ++ --
동등 관계자 : ==(같다), !=(다르다) < <= > >=
논리 연산자 : && || & | !
삼항 연산자 : ? :
할당 연산자 : = += -= *= /= %=

연산자 중 우선순위   ---> 처음부터 우선순위
(), (!, ++, --), (* / %), (+ -), (< <= > >= == !=), (&), (|), (&&), (||), (?), (= += -= *= /= %=)
