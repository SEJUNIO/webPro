[액션태그]
JSP 페이지에서 어떤 동작을 지시하는 태그
표준액션태그 - JSP 페이지에서 바로 사용할 수 있는 태그
커스텀액션태그 - 별도의 라이브러리를 설치해야만 사용할 수 있는 태그
표준액션태그와 커스텀액션태그는 태그 안에 사용하는 접두어가 다르기 때문에 외형상으로 쉽게 구분할 수 있다.
<JSP: ~> 표준액션태그임을 표시하는 접두어.
<C:SET ~> 커스텀 액션 태그 중 하나임을 표시하는 접두어.

액션태그 : 액션 태그는 XML 문법을 따르므로 단독 태그일 경우엔 반드시 '/>'로 끝나도록 만들어야 한다.
- 페이지 모듈화와 요청 흐름 제어 역할
- JSP페이지 내에서 어떤 동작을 하도록 지시하는 태그 

[forward, include, param]
forward - 현재의 페이지에서 다른 특정페이지로 전환할 때 사용 . 단 URL은 변하지 않음 / REQUEST 객체는 그대로.
﻿<% response.sendRedirect("sub.jsp");%>와 같은 비교 : 주소부분, ex1_forward.jsp?n=2로 실행하여 새로운 요청 객체(request객체)가 새롭게 생성

include - 현재 페이지에 다른 페이지를 삽입할 때 사용, 중복 영역을 처리하는데 사용.
<% include file ="copyright.html">과 유사

param - forward 및 include 태그에 데이터 전달을 목적으로 사용되는 태그.
