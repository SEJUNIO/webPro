[단일행 함수]

SQL함수는 단일행, 그룹 함수가 있으며
단일행함수는 오직 단일 행에서만 적용가능하고 행 별로 하나의 결과를 리턴 
문자, 숫자, 날짜, 형 변환 함수 등
[DUAL 테이블] 
듀얼 테이블은 테스트하기 위한 용도의 ORACLE 내장 객체, 한 행으로 출력된다.
주로 연산, 함수적용내용 확인시 활용. 산술연산결과를 한줄로 얻기 위해 오라클에서 제공하는 테이블
FLOOR - 소수점 아래를 버림
ROUND(X) - 특정 자릿수에서 반올림
ROUND(데이터, 반올림할 소수점 자리수) - 반올림할 자리수가 음수인 경우 -1은 십단위, -2는 백단위
TRUNC - N으로 지정한 자리수 이하를 버림

[문자 처리함수]
LOWER - 소문자로
UPPER - 대문자로
INITCAP - 첫 글자만 대문자로 나머지는 소문자로
CONCAT - 문자연결
SUBSTR - 문자를 잘라 추출 (한글은 1바이트), 시작위치가 음수이면 끝에서 자리수 센다.
SUBSTRB - 문자를 잘라추출(한글은 3바이트)
LENGTH - 문자길이 (한글은 1바이트)
LENGTHB - 문자길이 (한글은 2~3바이트)
LPAD, RPAD - 입력받은 문자열과 기호를 정렬하여 특정 길이의 문자열로 반환
TRIM - 잘라내고 남은 문자 표시

[날짜 함수]
SYSDATE - 시스템 저장된 현재 날짜 반환
MONTHS_BETWEEN - 두 날짜 사이가 몇 개월인지 반환
ADD_MONTHS - 특정 개월수를 더한다 
NEXT_DAY - 특정 날자에서 최초로 도래하는 인자로 받은 요일 날짜 반환
ROUND - 인자로 받은 날짜를 특정 기준 반올림
TRUNC - 인자로 받은 날짜를 특정 기준으로 버림

[형변환함수]  오라클을 사용하다 보면 숫자, 문자, 
날짜의 데이터 형을 다른 데이터형으로 변환해야 하는 경우가 생신다. 이럴때 사용하는 함수가 형변환 함수이다.

1.TO_CHAR : 날짜형 혹은 숫자형을 문자형으로 변환
ELECT SYSDATE, TO_CHAR(SYSDATE, 'YYYY-MM-DD') FROM DUAL;
YYYY 년도 4자리 표현 / YY 년도 2자리 표현
MM 월을 숫자로 표현 / MON 월을 알파벳으로 표현 / DD 일
DAY 요일 표현 / DY 요일을 약어로 표현
AM또는 PM / A.M또는 P.M.. 오전 오후 시각 표시

TO_CHAT(숫자데이터, ‘출력형식’)

TO_DATE : 문자형을 날짜형으로 변환 ( TO_DATE('문자', 'FORMAT')
오라클 기본 날짜 형식은  YY/MM/DD형식이므로 만약 년도를 4자리로 출력하려면 YYYY/MM/DD로 지정함

TO_NUMBER : 문자형을 숫자형으로 변환
TO_NUMBER는 특정 데이터를 숫자형으로 변환해 주는 함수입니다.
