[조건문]
[if문]
if(조건) {
	문장; 
}else(조건 이나 끝낼때) {
	문장;
}

[switch문]
﻿switch(비교할 값) {
case 값: 
	문장; break;
case 값: 
	문장; break;
Default:
	문장; break;
} //예시.
[삼항연산자]
﻿(불 표현식)? (참일 때 실행하는 문장) : (거짓일 때 실행하는 문장); //예시.

[짧은조건문]
﻿짧은 조건문은 논리연산자의 특성을 조건문으로 사용한다.
다음 논리합 연산자를 사용한 표현식은 뒤에 어떠한 값이 들어가도 항상 참이므로 우변을 실행하지 않는다.
ex// ﻿      true || OOO // 우변을 항상 실행하지 않는다.
	false || XXX //우변을 항상 실행
