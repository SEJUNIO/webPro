컴퓨터 프로그래밍 : 컴퓨터를 제어할 수 있는 명령어들의 집합
컴퓨터 프로그래밍 언어 : 컴퓨터 프로그램을 만드는 것.
자바  : OS 독립적 프로그래밍 언어
장점 : 독립적 OS, 객체지향 언어로 개발과 유지보수가 효율적으로 진행됨
단점 : 속도 저하 ( 현재는 딜레이 거의 없음)
자바 언어 탄생 배경 
1990년대 초 자바 탄생, 기존 C/C++에 비해 속도가 많이 느려 외면 받음
인터넷의 등장으로 인해 외면한 사람들에게 관심 얻기 시작
현재는 많은 웹 프로그래밍에 활용
자바 언어 특징
운영체제 영향을 받지 않음
개발자는 메모리를 관리 하지 않고 JVM이 메모리 자동관리 
다중 쓰레드 지원함
(자바 개발전 운영체제와 맞는 프로그램 설치 후 환경 변수 설정)
