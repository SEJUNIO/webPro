[제이쿼리]
제이쿼리는 자바 스크립트 라이브러리 중 가장 인기 있는 라이브러리. 이벤트 대응, 애니메이션과 메뉴효과 등에서 가장 단순하고 쉽게 페이지 제작을 할 수 있도록 도와줌.
- 문서 객체 모델과 관련된 처리를 쉽게 구현
- 일관된 이벤트 연결을 쉽게 구현
- 시각적 효과를 쉽게 구현
- ajax 애플리케이션을 쉽게 개발

예시 : $('선택자').함수();

[태그들]
html()  해당 선택자 안에 html코드 삽입 
text()   해당 선택자 안에 text 삽입 
hide()
show()
toggle()
val('변형될 value') ; value값 변형
css('속성', '속성값')
each ; 배열 처리함수 
addClass ; 클래스 추가
removeClass ; 클래스 삭제
append
prepend
attr 

[이벤트 관련 함수]
click(콜백함수), focus(콜백함수), etc...
off(), trigger()..

aJax관련 함수들 : $.get(), $.post(), $.ajax()
